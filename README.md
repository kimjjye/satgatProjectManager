# ✨ satgat project management system
김삿갓 프로젝트 관리 시스템


# ⚙️ Installation
> git clone <url>


### Client
> cd frontend


> npm run install


> npm run dev


### Server
> cd project-manager-backend


> ./mvnw spring-boot:run
